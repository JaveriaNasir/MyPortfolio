Hotel Registration

This MP application was basically developed as a GUI application that is used by a clerk at a hotel to register guests. When a guest arrives at the hotel, the clerk enters the name and address of the guest, what type of room the guest wants and then selects a room based on the desired type and availability. Point of Sale Registrar

This MP was asked to implement a point of sale register used by a clerk at a store to check out customers’ orders using SKU (Stock Keeping Unit) for each item. The application also maintains a register tape for all of the orders that have been entered into the application and stores that register tape in a file on disk.

Global Service Event Management (GSEM)

The Global Service Event Management (GSEM) is a capability that allows automatically transmitting Calls between Call Handling Systems (CHS) without manual intervention. Calls are posted by the Requester CHS and received by the Provider CHS. A set of transactions allows creating, updating, and calls closing The GSEM system ensures calls are passed to the Provider and keeps track of the call traffic. GSEM can handle internal (HP to HP) call passing requests, and external (Customers to HP, HP to Partners) call passing requests through a registration process.

  • Client: HP Enterprise
  • Date: Feb.2015- July.2015
  • Position: Senior Software Engineer
  • Service: Web Development
  • Technologies: C#.NET, VB.Net, SQL Server 2012, Visual Studio 2012
